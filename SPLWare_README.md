# SPLWare

**Composable data computing tools for reliable analytics.**

SPLWare builds practical, production-ready tools for data processing, analytics, and SQL generation.
Our projects focus on **clarity, correctness, and auditability** — helping teams trust their data pipelines.

## 🚀 Projects

### esProc

**A powerful data computing engine for complex analytics.**

esProc provides a flexible scripting language designed for data processing, ETL, and analytical workloads that are difficult to express in traditional SQL.

### SQLazy

**Write complex SQL step-by-step in natural language — then compile it into auditable SQL.**

SQLazy turns AI-assisted query building into a transparent and verifiable process, eliminating black-box SQL generation.

👉 [https://github.com/SPLWare/sqlazy](https://github.com/SPLWare/sqlazy)

## 🎯 What We Believe

Modern analytics faces two major problems:

* SQL becomes increasingly complex and hard to maintain
* AI can generate SQL that runs — but not always SQL that is correct

SPLWare tools aim to bridge this gap by combining:

* Human-readable logic
* Compiler-level guarantees
* Production-grade reliability

---

## 🌱 Why SPLWare

We believe data tools should be:

* **Transparent** — no black boxes
* **Composable** — step-by-step logic building
* **Auditable** — easy to review and verify
* **Practical** — designed for real production use
